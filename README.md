# TimeTracking für macOS — Releases

Öffentliches Verteilungs-Repo für die TimeTracking-macOS-App (Solutions2Share).

- Jedes Release enthält eine **Developer-ID-signierte und von Apple notarisierte** `TimeTracking.dmg`.
- Die App prüft dieses Repo selbstständig auf Updates (Einstellungen → Updates) und installiert neue Versionen direkt aus der App.
- Releases werden automatisch vom CI-Workflow des TimeManager-Repos veröffentlicht (`v*`-Tags).

**Installation:** DMG laden, öffnen, App in den Ordner „Programme" ziehen.
